# cliferay curl new-versioned-object

Create a new versioned object definition with some entries  
  
This requires the following feature flags:  
  
  feature.flag.LPD-17564=true  
  feature.flag.LPD-11235=true  
  feature.flag.LPD-34594=true  


## Usage

```bash
cliferay curl new-versioned-object
```
