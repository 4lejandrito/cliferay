# cliferay weblogic

Start a Liferay bundle on WebLogic 14.1.2  
  
Liferay will be available at http://localhost:7001.  
The debugger will be available on port 5005.  
  
Run inside an empty folder the first time. After that, it is idempotent.  


## Usage

```bash
cliferay weblogic VERSION
```

## Examples

```bash
cliferay weblogic dxp/2025.q1.12-lts
```

## Arguments

#### *VERSION*

The version to run

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes
