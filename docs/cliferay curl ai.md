# cliferay curl ai

Run a curl command based on a prompt

## Usage

```bash
cliferay curl ai PROMPT [OPTIONS]
```

## Examples

```bash
cliferay curl ai "create a sample blog posting"
```

```bash
cliferay curl ai "list all object definitions"
```

## Environment Variables

#### *OPENAI_API_KEY*

Your OpenAI API Key

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

## Arguments

#### *PROMPT*

The prompt

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

## Options

#### *--generate, -g*

Output the command instead of running it
