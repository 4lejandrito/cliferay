# cliferay bundle

Set up bundle from https://releases.liferay.com  
  
The bundles directory will be replaced by the downloaded one  
The commit inside the .githash file will be checked out in the repository  


## Usage

```bash
cliferay bundle VERSION
```

## Examples

```bash
cliferay bundle dxp/2023.q4.0
```

## Arguments

#### *VERSION*

The version to run
