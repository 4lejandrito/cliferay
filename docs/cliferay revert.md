# cliferay revert

Revert commits that match a text

| Attributes       | &nbsp;
|------------------|-------------
| Alias:           | r

## Usage

```bash
cliferay revert TEXT [OPTIONS]
```

## Arguments

#### *TEXT*

Text to filter commits by (e.g., LPD-1234)

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

## Options

#### *--branch, -b BRANCH*

Source branch
