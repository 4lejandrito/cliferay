# cliferay team assigned

Show all tickets ever assigned to the user

## Usage

```bash
cliferay team assigned [USER]
```

## Examples

```bash
cliferay team assigned alejandro.tardin
```

## Environment Variables

#### *JIRA_API_USER*

Your Jira email

| Attributes      | &nbsp;
|-----------------|-------------
| Required:       | ✓ Yes

#### *JIRA_API_TOKEN*

| Required:       | ✓ Yes
