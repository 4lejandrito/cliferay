# cliferay docker

Start a Liferay Docker container

## Usage

```bash
cliferay docker [IMAGE]
```

## Examples

```bash
cliferay docker
```

```bash
cliferay docker portal:7.4.3.132-ga132
```

## Arguments

#### *IMAGE*

The Docker image name and optional tag to run

| Attributes      | &nbsp;
|-----------------|-------------
| Default Value:  | portal
