# cliferay curl

Send HTTP requests to the server

## Usage

```bash
cliferay curl COMMAND
```

## Commands

- [new-instance](cliferay%20curl%20new-instance.md) - Create a new instance
- [new-api](cliferay%20curl%20new-api.md) - Create a new API Builder API
- [new-object](cliferay%20curl%20new-object.md) - Create a new object definition with some entries
- [new-versioned-object](cliferay%20curl%20new-versioned-object.md) - Create a new versioned object definition with some entries
- [new-site-object](cliferay%20curl%20new-site-object.md) - Create a new site scoped object definition with some entries
- [batch-export](cliferay%20curl%20batch-export.md) - Export site pages with batch
- [batch-import](cliferay%20curl%20batch-import.md) - Import a new object entry with batch
- [mcp-developer-prompt](cliferay%20curl%20mcp-developer-prompt.md) - Add a developer prompt to the MCP Server
