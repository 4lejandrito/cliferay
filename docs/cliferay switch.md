# cliferay switch

Switch to a different bundle  
  
This command automates switching between different Liferay bundles directories, each with its own database.  


## Usage

```bash
cliferay switch NAME
```

## Examples

```bash
cliferay switch LPD-XXXXX
```

```bash
cliferay switch master
```

## Arguments

#### *NAME*

The name of the bundle to switch to  
  
If a bundle with that name is not present it will be copied from the current one.
