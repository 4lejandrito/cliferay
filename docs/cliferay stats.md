# cliferay stats

Calculate various Git statistics

## Usage

```bash
cliferay stats COMMAND
```

## Commands

- [users](cliferay%20stats%20users.md) - Get information about users
- [years](cliferay%20stats%20years.md) - Show commit count per year
- [ranking](cliferay%20stats%20ranking.md) - Show commit ranking
- [tickets](cliferay%20stats%20tickets.md) - Show tickets worked on, based on commits
- [commits](cliferay%20stats%20commits.md) - Print the commits for the user
- [folders](cliferay%20stats%20folders.md) - Print the folders changed by the user
- [review](cliferay%20stats%20review.md) - Write a review based on Git and Jira stats
- [assigned](cliferay%20stats%20assigned.md) - Show all tickets ever assigned to the user
- [headless-summary](cliferay%20stats%20headless-summary.md) - Show tickets grouped by folder
