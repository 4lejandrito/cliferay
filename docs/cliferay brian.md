# cliferay brian

Forward an existing PR to Brian and close it

## Usage

```bash
cliferay brian PR
```

## Examples

```bash
cliferay brian https://github.com/liferay-headless/liferay-portal/pull/1433
```

## Arguments

#### *PR*

Link to the GitHub PR
